# The Impact of Daily Habits on Mental Health

A data analysis project that investigates how daily lifestyle
habits such as sleep, exercise, screen time, and diet quality
affect mental health outcomes (mood score and stress level)
across 2,000 individuals.

## Research Question
Do daily routine variables (sleep, exercise, diet, and digital
habits) significantly influence an individual's mood and stress levels?

## Dataset
- 2,000 participants
- Variables: sleep hours, screen time, exercise, fatigue,
  diet quality, caffeine intake, pending tasks, weather
- Target variables: mood score, stress level, fatigue level

## Analysis Includes
- Data cleaning and preprocessing
- Descriptive statistics (mean, min, max, std)
- Distribution analysis with histograms
- Correlation analysis between lifestyle habits and mental health
- Diet quality and weather distribution analysis
- Visualizations using Matplotlib and Seaborn

## Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy)
- Jupyter Notebook
- Google Colab
